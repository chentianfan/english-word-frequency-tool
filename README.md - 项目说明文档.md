# 🐰 英语单词词频统计工具 Pro

> 一个功能强大的英语单词词频统计与分析工具，支持多格式文档、离线词典、难度分级、词根词缀分析、批量处理及多种格式导出。

## ✨ 功能特点

### 📊 核心功能
- ✅ **词频统计** - 快速统计文档中所有英语单词的出现频率
- ✅ **中文释义** - 支持离线词典（77万+词条）和在线翻译双模式
- ✅ **难度分级** - 自动标注单词难度（中考/高考/四级/六级/考研/雅思/托福/GRE）
- ✅ **词根词缀** - 智能分析单词的词根词缀组成
- ✅ **词形还原** - 自动还原动词时态、名词复数、形容词比较级等
- ✅ **停用词过滤** - 可过滤 the/of/and 等无意义停用词

### 📁 文件格式支持
- ✅ **TXT 文本文件**
- ✅ **PDF 文档**
- ✅ **Word 文档** (.docx)

### 📤 导出格式
- ✅ **Excel** (.xlsx) - 带打印优化，方便直接打印
- ✅ **CSV** (.csv) - 通用表格格式
- ✅ **Anki 生词本** (.txt) - 可直接导入 Anki 背单词

### 🎨 用户体验
- ✅ **图形界面 (GUI)** - 简洁美观，操作简单
- ✅ **多语言界面** - 支持中文/英文一键切换
- ✅ **多线程翻译** - 加速翻译过程
- ✅ **断点续传** - 翻译中途退出可继续
- ✅ **批量处理** - 支持整个文件夹批量处理
- ✅ **配置保存** - 自动记住你的设置偏好
- ✅ **实时进度条** - 清晰显示处理进度

## 🚀 快速开始

### 方式一：直接使用 exe（推荐，无需安装 Python）

1. 下载最新的 `词频统计工具.exe`
2. 和 `ecdict.csv`（词典文件）放在同一个文件夹
3. 双击运行即可

### 方式二：Python 源码运行

#### 环境要求
- Python 3.8 或更高版本

#### 安装依赖
```bash
pip install openpyxl requests python-docx PyPDF2
```

#### 运行程序
```bash
python word_frequency_analyzer_gui.py
```

## 📖 使用教程

### 第一步：选择输入文件
- 点击「浏览...」选择要分析的文件
- 支持 .txt、.pdf、.docx 格式
- 勾选「批量处理模式」可以处理整个文件夹

### 第二步：设置分析选项
- **忽略停用词**：过滤 the/of/and 等常见词
- **词形还原**：把 went/goes/going 都还原成 go
- **纯离线模式**：只用本地词典，不联网
- **获取中文释义**：是否翻译单词
- **显示难度等级**：显示单词属于哪个考试级别
- **显示词根词缀**：分析单词的词根词缀
- **只保留有释义的单词**：过滤掉查不到的生僻词

### 第三步：选择导出格式
- Excel (.xlsx) - 推荐，格式美观，可直接打印
- CSV (.csv) - 通用格式，适合导入其他软件
- Anki 生词本 - 适合导入背单词软件

### 第四步：开始分析
- 点击「开始分析」按钮
- 等待进度条完成
- 分析完成后会自动打开输出文件夹

## 📚 关于离线词典

本项目使用 [ECDICT](https://github.com/skywind3000/ECDICT) 开源英汉词典数据：

- **完整版**：77万+ 词条，63MB
- **精简版**：4.2万 常用词，8MB（推荐日常使用）

> 💡 提示：如果没有词典文件，程序会自动降级为在线翻译模式，需要联网。
